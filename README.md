# FE-Interview

## 前言

整理这份面试题的目的是为了让更多前端`(当然，还有我自己)`能够在面试中抓住面试官想要得到的知识点，从而给面试官一个更满意的答案。在面试中我并不倡导使用套路来套路面试官，短短的几个小时的面试本质上也并不能看出面试者的实际水平，但是不可否认的是掌握足够的技巧和经验是可以为面试者拿到一份满意的 Offer 的。一些是自己遇到的还有看掘金大佬发布的、还有一些朋友面试的我都会在里面写到
